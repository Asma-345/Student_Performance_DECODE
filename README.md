# Student Performance Prediction

## Project Overview
This project analyzes student data and predicts their final scores using **Linear Regression**. It demonstrates the complete data science pipeline: data cleaning, exploratory analysis, feature engineering, model building, and evaluation.

## Steps Performed
1. **Data Loading** – Imported dataset and checked structure.
2. **Data Cleaning** – Handled missing values, removed duplicates, and corrected inconsistent data.
3. **Exploratory Data Analysis (EDA)** – Visualized data relationships and distributions to understand patterns.
4. **Feature Engineering** – Encoded categorical variables and created meaningful features.
5. **Model Training & Evaluation** – Built Linear Regression model and evaluated using **R²** and other metrics.
