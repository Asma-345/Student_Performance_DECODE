# Student Performance Prediction (DECODE Project)

This project predicts student Final_Score based on academic and demographic features. 

## Steps
1. Load & inspect data
2. Clean & preprocess
3. EDA & visualizations
4. Feature engineering & model pipeline
5. Model evaluation & interpretation

## Model Performance
- R² = 0.59
- MAE = 2.45
- RMSE = 3.

The model identifies Midterm, Attendance, and Assignments as key predictors.
